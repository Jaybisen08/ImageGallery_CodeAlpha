## ✨ About The Project

**SpaceVerse** is a modern, responsive, and visually immersive image gallery designed to take users on a journey through the cosmos.

From glowing galaxies to mysterious black holes, SpaceVerse creates an engaging digital exploration of space with smooth animations, category filters, and fullscreen image previews.

Built as part of the **CodeAlpha Frontend Development Internship**.

---

## 🎥 Features

✨ **Cinematic Landing Page**
A fullscreen animated video background for an immersive first impression.

🖼️ **Interactive Space Gallery**
Explore 14 stunning HD space images across multiple categories.

🎯 **Smart Category Filters**
Filter images instantly by category.

🔍 **Lightbox Preview**
View images in fullscreen with next/previous navigation.

⚡ **Smooth Animations**
Hover effects, transitions, and interactive UI elements.

📱 **Fully Responsive Design**
Optimized for desktop, tablet, and mobile devices.
